# Model dosyaları

Bu klasöre LLM model dosyasını (`.gguf`) koyun.

**Önerilen:** `qwen2.5-1.5b-instruct-q4_k_m.gguf`  
(Hugging Face veya başka bir kaynaktan indirip buraya kopyalayın.)

`chat.py` varsayılan olarak bu isimdeki dosyayı arar; farklı bir model kullanacaksanız `models/` içinde en az bir `.gguf` bulunması yeterli (ilk bulunan kullanılır).
